# TP 2 : Adressage IP et Sous-réseaux (Subnetting)

## Objectifs du travail
Ce travail pratique a pour but de maîtriser le découpage de réseaux IPv4 (Subnetting). Les objectifs spécifiques sont :
- Calculer les nouveaux masques de sous-réseau.
- Déterminer les plages d'adresses pour chaque sous-réseau.
- Calculer le nombre d'hôtes disponibles par sous-réseau.

## Technologies utilisées
- **Adressage IPv4** : Manipulation d'adresses et de masques binaires/décimaux.
- **Calculs binaires** : Division de plages d'adresses (VLSM/FLSM).

## Contenu du dépôt
Ce dépôt contient :
1. Les calculs détaillés des divisions de réseaux.
2. Un rapport technique au format PDF (source LaTeX incluse).

## Instructions
Pour compiler le rapport LaTeX :
1. Assurez-vous d'avoir une distribution TeX installée (TeX Live, MiKTeX).
2. Exécutez la commande : `pdflatex rapport_tp2.tex`.
